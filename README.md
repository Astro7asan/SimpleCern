SimpleCern 🌌

An Interactive Particle Physics Guide & Experimental Suite

Created by Hasan Alhashmi

Welcome to SimpleCern, an educational high-energy physics dashboard designed and curated by Hasan Alhashmi. Based on the academic guide "The Architecture of Reality.docx", this single-page interactive application brings the complex world of particle accelerators, subatomic structures, and quantum fields to life through real-time simulations, animated visual models, and tactile experiments.

🚀 Core Features

1. LHC Ring & Cavern Explorer

Synchrotron Map: An interactive circular SVG layout of the 27-kilometer subterranean Large Hadron Collider ring spanning the France-Switzerland border.

Beam Velocity Control: Move the slider up to $99.999999\% \ c$ to watch the opposing proton orbits scale in frequency and timing.

Cavern Diagnostics: Select and examine CERN's four primary collision points:

ATLAS: General-purpose particle camera tracking Higgs Boson decay tracks.

CMS: Solenoidal detector providing validation checks for physical breakthroughs.

ALICE: Smasher of heavy Lead (Pb) ions recreating primordial quark-gluon plasma soup.

LHCb: Spectrometer tracking asymmetry in beauty (bottom) quarks to investigate the matter-antimatter imbalance.

Live Science Portal: Clicking each cavern dynamically populates direct telemetry links to official scientific directories on home.cern.

2. Standard Model Cabinet Matrix

Subatomic Dossier: Access a modular particle grid organizing the fundamental blocks of reality (Quarks, Leptons, Gauge Bosons, and the Scalar Higgs Boson).

Tactile Cards: Click on any particle to retrieve its physical characteristics:

Fractional electric charges (e.g., $+2/3e$, $-1/3e$).

Rest masses (from ultra-lightweight neutrinos up to the heavy $173 \ \text{GeV/c}^2$ Top Quark).

Spin alignments, generation numbers, conservation parameters, and their distinct roles in cosmic architecture.

3. Baryon Construction Lab

Interactive Workbench: Construct nucleons directly in your browser. Drag or select Up ($u$) and Down ($d$) quarks into active slot registries.

Dynamic Gluon Bonds: Generates pulsing, curvilinear vector paths representing localized color-charge interactions and high-energy gluon exchanges.

Nuclear Chemistry Analytics: Performs real-time fractional charge math to evaluate baryon states, detecting and verifying:

Protons ($uud = +1e$) — Stable hydrogen-core nuclei.

Neutrons ($udd = 0e$) — Co-nucleons undergoing beta decay if left unbonded.

Delta Isobars ($uuu = +2e$, $ddd = -1e$) — Unstable strong-decay states.

4. Cosmic Higgs Snowfield Engine

Interactive Particle Drag Metaphor: An HTML5 <canvas> simulation demonstrating Peter Higgs' theory of inertia.

Visual Legend: Watch how different particles interact with the medium:

Photons (The Skiers): Avoid interacting with the medium, remaining massless and traveling at pure light-speed.

Electrons (The Walkers): Encounter minor resistance, representing light atomic mass.

Top Quarks (Heavy Boots): Face extreme drag, simulating their massive inertial weight.

Medium Density Modulator: Dynamically scale the Higgs vacuum density parameter to observe its immediate effect on mass accumulation and particle acceleration.

5. Tabletop Linear Accelerator

Digital Twin Simulation: Modeled after physical classroom apparatus (recorded under photo directory IMG_6041.jpg).

Electromagnetic Coil Timing: Fire proton bunches (represented by steel balls) through copper-coil electromagnets ($L_1, L_2, R_1, R_2$). Switch between automated timing grids or manual alignments to maximize collision impact velocity.

Conservation of Momentum Analytics: After head-on collisions, the vector tracker monitors asymmetrical energy balances. Displays values for Missing Transverse Momentum ($\vec{p}_T$) and Missing Transverse Energy ($\not\!\!E_T$) to detect escaping, non-interactive neutral neutrinos ($\nu$).

6. The 125 GeV "Discovery Bump" Chart

Live Collision Plotter: Recreates the landmark discovery announced on July 4, 2012.

Slam Proton Beams: Simulate millions of proton collisions to see background noise accumulate along a sloping exponential baseline curve.

Statistical Analysis Peak: Toggle the "Reveal Higgs Peak" button to witness the historic $125 \ \text{GeV}$ statistical bump stand out from standard background noise at the 5-sigma gold standard.

🛠️ Installation & Getting Started

Because SimpleCern was designed with high accessibility and portability in mind, it is packaged as a fully self-contained, single-file application. There are no node modules to install, no complex dependencies, and no servers to configure.

Run Locally:

Clone the Repository:

git clone https://github.com/your-username/simplecern.git
cd simplecern


Open in Browser:
Simply double-click the physics_guide.html file or run it via a simple local python server:

python3 -m http.server 8000


Now, navigate to http://localhost:8000/physics_guide.html inside any modern browser.

💻 Tech Stack

Markup: HTML5 (semantic layout)

Visualizations: Dynamic SVG path rendering & HTML5 Canvas Context 2D API
